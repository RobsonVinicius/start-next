## Getting Started

Install the dependencies: 

Next Project:

```bash
npm install
# or 
yarn install
```
Run the development server:
```bash
npm run dev
# or
yarn
yarn dev
```

### NOTA:

```
* Utilizar o Next na versão superior a "next": "10.0.3" para evitar o problema com o Build "Loader"

```
